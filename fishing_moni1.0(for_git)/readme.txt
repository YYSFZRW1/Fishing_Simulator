命令行钓鱼模拟器
一款纯文本的命令行钓鱼游戏，玩家通过抛竿、输入 WASD 按键序列来捕获不同种类和重量的鱼。游戏支持多种难度，挑战你的反应速度！
（本游戏不联网，不用到数据库，点击start.bat即可一键运行！输入help随时查看帮助！）

（本游戏暂时只支持Windows，Linux系统后续会开发，其它系统不会（┑(￣Д ￣)┍） ）


游戏简介

目标：通过抛竿捕获鱼，完成 WASD 按键挑战来收集稀有鱼。
难度：简单、中等、高级、“你抓不到我”，难度越高，鱼越重，时间越短。
玩法：输入命令抛竿，鱼上钩后在限定时间内按顺序输入 WASD 序列。
特色：
多种鱼类（普通、稀有、传说、unbelievable（全图鉴解锁））。
高难度下鱼逃跑会嘲讽你，可不要被气坏了！
未来计划：鱼竿系统，捕鱼失败可能掉落鱼竿，提升时间限制。

运行游戏：双击打开start.bat即可

鼠标左键双击打开start.bat，进入游戏界面。
按提示选择难度（1-4）并开始游戏（难度5需全图鉴解锁）。

命令说明
cast：抛竿开始钓鱼（支持任意大小写，如 CAST, cast）。
inventory：查看已捕获的鱼（支持任意大小写）。
exit：退出游戏（支持任意大小写）。
更多命令启动后输入help查看~~
玩法规则

选择难度：
1（简单）：轻鱼，时间充裕，适合新手。
2（中等）：中等鱼，适中挑战。
3（高级）：重鱼，时间较短。
4（你抓不到我）：超重鱼，极短时间，失败可能被嘲讽！
5 全图鉴解锁：隐藏鱼等你来抓！

抛竿：
输入 cast，等待 1-3 秒，鱼会上钩。

捕鱼挑战：
鱼上钩后，显示鱼的种类、重量和 WASD 序列。
在时间限制内逐个输入 WASD（支持 W/w, A/a, S/s, D/d）。
正确输入捕获鱼，错误或超时鱼逃跑。

查看背包：
输入 inventory，列出捕获的鱼（种类、名称、重量）。

退出：
输入 exit 结束游戏。

注意事项
输入提示：
捕鱼时只输入 W, A, S, D（大小写均可），避免按 Enter、空格或其他键。
命令输入支持任意大小写，忽略空格（如 " cast " 有效）。

时间限制：
高难度（如“你抓不到我”）时间极短，需快速反应。
建议练习低难度熟悉 WASD 输入。

Windows 用户：
默认使用 msvcrt 处理输入，无需额外安装。
建议通过命令行（CMD/PowerShell）运行，避免 IDE 终端输入异常。

已知问题：
在某些 IDE（如 VS Code）终端运行可能导致输入丢失，推荐命令行运行。
若输入未被捕获，请检查键盘缓冲区或运行环境。


未来计划

鱼竿系统：捕鱼失败时低概率掉落鱼竿，不同品质的鱼竿延长 WASD 输入时间。
积分系统：根据鱼的种类和重量计算分数。
更多鱼类：增加鱼的种类和特殊效果。
保存记录：将捕获的鱼保存到文件。
